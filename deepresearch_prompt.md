# 依頼内容
〇〇の検索調査をお願いします。

### 調査概要

### 制約事項
- 査読済み論文だけでなく、学会発表、学位論文、政府・自治体・環境保護団体などの調査報告書も対象とする。
- 過去10～20年の研究を優先するが、それ以前の重要な研究も対象とする。
- 系統的レビュー論文やメタ分析研究が存在する場合、それを優先的に参照すること。
- 英語または日本語の文献を参照すること。
- 日本語で報告すること。
- 最後に文献情報（文献タイトル / 出版年 / 概要 / 参照リンク）をまとめて表で報告すること。

### 出力方法
- 日本語で出力してください。
- マークダウン(.md)にて出力してください。
- 表形式でのまとめを入れてください（必須の列: ）。
